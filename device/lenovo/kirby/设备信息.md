## 下一步操作

1. **获取分区大小**：
   ```bash
   adb shell lsblk
   adb shell getprop | grep partition
   ```

2. **获取显示信息**：
   ```bash
   adb shell wm size
   adb shell wm density
   ```

3. **提取专有文件**：
   ```bash
   cd device/lenovo/kirby
   ./extract-files.py
   ```

4. **更新设备树配置**：
   - 根据实际分区大小更新 BoardConfig.mk
   - 根据实际硬件信息更新 device.mk
   - 更新构建指纹等信息
